## Requirement

- node 20.10.0
- npm 10.2.5

## how to use

1. テンプレートのダウンロード

- git clone "git@github.com:yasu-pro/lp_coding_temp.git"

2. 新規プロジェクトの作成

- mv lp_coding_temp/ [新規プロジェクト名] && cd ./[新規プロジェクト名] && code . && npm install

3. リポジトリを変更する

- git remote set-url origin [変更する GitHub のリポジトリ]

4. 開発(volta 使用)

- npm run dev

5. build

- npm run build
